Documentation: [Found Here](https://github.com/josephmancuso/python-framework/wiki)

#### Installation:

```
    $ git clone http://github.com/josephmancuso/python-framework.git
    $ python craft install
    $ python craft serve
```

Go to `http://127.0.0.1:8000/`
